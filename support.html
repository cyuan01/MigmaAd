How to get started

1. Use any computer to upload advertising materials onto a USB stick. PNG, JPG, and GIFF formats are all accepted. If you are having trouble uploading the files, look to the support manual for your specefic computer brand and model.

2. Take the USB stick and plug it into one of the USB ports on the MigmaAd device.

3. Using an HDMI cord, connect MigmaAd to the monitor.

4. Plug MigmaAd into a power outlet

5. Allow MigmaAd to start up, save your files, and then watch your advertisements play!

Unable to see images?

Sometimes images are saved without their file extension. For example, an image of file type "jpeg" and name "image" would normally be saved on your computer as "image.jpeg" but in small cases it is saved as just "image".


Follow these steps to troubleshoot:


1. Check file type by right clicking the image. If on a mac, go to the "get info" tab and if on a pc, go to the "properties" tab to see the kind of file.

2. Rename the image file with the file type by renaming it "name.jpeg" or "name.png" depending on the kind of file you have from step 1. and the original "name" of the image.

3. Save the images to your USB stick and restart MigmaAd with the new USB stick.
